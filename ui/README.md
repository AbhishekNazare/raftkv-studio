# RaftKV Studio UI

React/TypeScript observability dashboard for the RaftKV control plane.

## Run

Start the control plane:

```bash
./scripts/run_control_plane.sh
```

Start the UI:

```bash
cd ui
npm run dev
```

Open:

```text
http://127.0.0.1:5173
```

## Build

```bash
cd ui
npm run build
```
